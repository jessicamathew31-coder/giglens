# GigLens — India's Gig Worker Financial Health Audit Engine

A production-grade financial analytics project built to quantify the financial vulnerability of India's 23.5 million gig workers.

## Live Demo
Coming soon on Vercel

## What It Does
- Calculates a personalised GigLens Score (0-100) across 5 financial dimensions
- Interactive dashboard with platform and city-level risk analysis
- Downloadable financial health report
- Built on real data from PLFS, NITI Aayog, and Fairwork India 2024

## Tech Stack
- Frontend: React.js (deployed on Vercel)
- Data Analysis: Python (pandas, matplotlib)
- Database: SQLite
- Dataset: 500 synthetic gig workers calibrated to real published data

## Scoring Dimensions
- Income Stability (25%) — vs city minimum wage
- Expense Burden (20%) — expenses as % of earnings
- Benefits Gap (20%) — insurance and savings coverage
- Crisis Resilience (20%) — months of expenses in savings
- Platform Risk (15%) — Fairwork India 2024 scores
